# 任务打卡 - 产品需求说明

## 1. 概述与目标

**应用名称**：任务打卡
**类型**：Android 本地任务管理应用
**核心价值**：帮助用户管理每日待办，通过打卡勾选完成任务，并自动重置。

---

## 2. 功能模块

### 2.1 任务管理

| 功能 | 描述 |
|------|------|
| 添加任务 | 底部 "+" 按钮 → 弹出底部输入框 → 输入名称 → 确认生成 |
| 编辑任务 | 点击任务文字 → 进入内联编辑模式 → 光标定位 → 保存/取消 |
| 删除任务 | 右滑显示删除按钮 → 确认弹窗；或编辑模式下批量勾选删除 |
| 排序 | 长按拖动把手排序（手动）；或设置按创建时间/完成状态自动排序 |

### 2.2 打卡交互（核心）

| 功能 | 描述 |
|------|------|
| 勾选切换 | 点击 CheckBox → 切换已完成/未完成 |
| 视觉反馈 | 勾选后：文字加删除线 + 颜色变灰(#9E9E9E) + 背景微变 |
| 全选/重置 | 底部提供 "全选" / "取消全选" / "重置今日" 按钮 |
| 每日重置 | 每天凌晨自动取消所有勾选，任务列表保留 |

### 2.3 数据持久化

| 功能 | 描述 |
|------|------|
| 本地存储 | Room Database，字段：id, title, isCompleted, orderIndex, createdAt |
| 状态同步 | 每次操作实时写入，关闭应用再打开状态不丢失 |
| 历史归档 | "已完成" 页面展示所有打卡记录（时间戳），支持一键清空 |

---

## 3. UI/UX 设计

### 3.1 主页布局

```
┌─────────────────────────────────┐
│  📋 任务打卡            ⋮菜单  │  ← 标题栏
├─────────────────────────────────┤
│  ☐ 买菜                    ≡   │  ← 任务条目（未完成）
│  ☑ 跑步 ───────────────── ≡   │  ← 任务条目（已完成，有删除线）
│  ☐ 读书                    ≡   │
│  ☐ 喝水                    ≡   │
├─────────────────────────────────┤
│  [全选]  [取消全选]  [重置今日] │  ← 底部快捷操作栏
└─────────────────────────────────┘
            [  ＋  ]                ← 悬浮添加按钮
```

### 3.2 任务条目结构

- **左侧**：Material Design CheckBox
- **中间**：任务名称（可点击编辑）
- **右侧**：拖拽把手（三条横线图标） + 更多菜单（三点图标）

### 3.3 编辑交互

- 点击任务文字 → 变为 EditText 内联编辑，光标出现
- 输入框置空 → 弹出确认："删除此任务？"
- 键盘确认后自动保存

### 3.4 主题与反馈

- **深色/浅色模式**：跟随系统设置
- **触感反馈**：CheckBox 点击时 HapticFeedback
- **动画**：勾选时 CheckBox 缩放动画（150ms）

## 4. 技术实现要求

### 4.1 技术栈

| 项目 | 要求 |
|------|------|
| 语言 | Kotlin |
| 最低 SDK | 24 (Android 7.0) |
| 目标 SDK | 34 (Android 14) |
| 架构 | MVVM + StateFlow |
| 数据库 | Room |
| UI | Material Design 3 + RecyclerView |
| 依赖注入 | 手动（无 Hilt，保持轻量） |

### 4.2 数据库设计

**Task 实体：**
```
id: Long (Primary Key, AutoGenerate)
title: String (任务名称)
isCompleted: Boolean (打卡状态)
orderIndex: Int (排序权重，越小越靠前)
createdAt: Long (创建时间戳，毫秒)
```

### 4.3 关键实现

- **拖拽排序**：RecyclerView + ItemTouchHelper
- **每日重置**：WorkManager 定时任务（每天凌晨 00:01 执行）
- **历史归档**：另一个 Room 表 `TaskHistory`，记录完成时间
- **无网络权限**：纯本地应用

---

## 5. 页面清单

1. **主页面**（TaskListActivity）
   - 任务列表 RecyclerView
   - 底部快捷操作栏
   - 悬浮添加按钮

2. **已完成归档页**（HistoryActivity）
   - 所有已完成任务列表
   - 一键清空按钮

3. **底部弹窗**
   - 添加任务输入框
   - 编辑任务输入框
   - 删除确认对话框
